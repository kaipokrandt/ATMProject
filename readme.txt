ATM Project simulating a client-server interaction where a user is able to deposit, withdraw, and check balance.

How to RUN:

Download server.py and client.py 
Run command "python3 server.py" - you should see

"Server Listening on port 9999..."

Then, **IN A SEPERATE TERMINAL** run "python3 client.py" - you should see

"Starting client..."
"Connected to server."

"Options:"
"1. Check Balance"
"2. Deposit Money"
"3. Withdraw Money"
"4. Exit"
"Enter your choice: __"
